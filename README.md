# LMS_Proprty_App_backend
Overview: 
This Real Estate CRM Portal is a comprehensive lead-to-deal management system 
designed specifically for real estate companies selling builder properties. The system 
manages the complete sales cycle from lead capture through Facebook Pixel to deal 
closure, with multirole access for Admins, Sales Executives, Sales Coordinators, and 
Tele Callers. 

System Goals: - Capture leads from multiple sources (Facebook, Instagram, Google, website, etc.) - Automatically map leads to area and assign to active projects only - Track all lead activities: calls, visits, follow-ups - Automate reminders, follow-up notifications, and maintain logs - Implement sub-modules: VCVP (Visit Proposed/Confirmed), Site Visit, Cross-Pitch - Provide dashboards and reports for Admins and Executives - Enable responsive and mobile-friendly user interface
